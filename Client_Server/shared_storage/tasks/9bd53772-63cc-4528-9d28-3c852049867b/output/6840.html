四、 综合分析结论

综合分析红外光谱数据，该谱图呈现出典型的长链脂肪族聚合物底骨架与酯类侧链基团的混合特征。 首先，谱图在2916 cm-1与2848 cm-1处出现强烈的吸收峰，这分别是亚甲基（-CH2-）的反对称与对称伸缩振动，配合1467 cm-1处的-CH2-面内弯曲振动以及717 cm-1处的长链亚甲基面内摇摆振动特征峰，充分证明了体系中存在大量的连续-(CH2)n-长碳链结构，这是聚乙烯骨架的典型特征。 其次，谱图在1737 cm-1处出现尖锐且极强的吸收峰，归属于酯类羰基（C=O）的伸缩振动；同时在1240 cm-1和1021 cm-1处观察到酯基C-O-C的不对称和对称伸缩振动强峰。此外，1371 cm-1处的吸收峰指示了由于酯基连接的甲基（-CH3）产生的对称弯曲振动。这些特征完美对应了醋酸酯基团（即乙酸乙烯酯链段）的存在。 综上所述，无论是聚乙烯长碳链骨架的振动特征，还是侧链醋酸酯基团的强吸收信号均非常清晰且完整。由此可以判定，该物质为乙烯与醋酸乙烯的共聚物（EVA），且未见其它明显杂质峰，判断为高弹性的纯EVA聚合物体系。
